A bucket for [Scoop](https://scoop.sh), the Windows command-line installer.

## Using the bucket

---------------------------------

To add this bucket, run `scoop bucket add lava https://github.com/moposx/lava-bucket`.

To install, do `scoop install <manifest>`.
